This project is a web-enabled book recommendation system designed to help users discover books tailored to their interests and preferences. The application provides an intuitive and engaging platform for users to search for books, explore trending titles, and receive personalized recommendations based on their input.

Project Structure
client/ — Frontend application
server/ — Backend API built with Node.js and Express

To run the project locally, follow these steps:
1. Install Dependencies for Both Client and Server
Open a terminal and navigate to the `client` folder:
	cd client
	npm install	
Open another terminal window, navigate to the `server` folder:
	cd server
	npm install
2. Start the Backend Server
In the terminal for the `server` folder, start the backend:
	npm start
