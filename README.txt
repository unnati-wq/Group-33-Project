This project is a web-enabled book recommendation system designed to help users discover books tailored to their interests and preferences. The application provides an intuitive and engaging platform for users to search for books, explore trending titles, and receive recommendations based on overall user ratings.

Project Structure
client/ — Frontend application
server/ — Backend API built with Node.js and Express
milestone3.py - code used for cleaning, wrangling, and ingesting the data
list_of_dependencies.txt - A list of dependencies
README.txt - A README file consisting of instructions on how to run your project locally and a short description of your project as well

To run the project locally, follow these steps:
1. Install Dependencies for Both Client and Server
Open a terminal and navigate to the `client` folder:
	cd client
	npm install	
Open another terminal window, navigate to the `server` folder:
	cd server
	npm install
2. Start the Backend Server
In the terminal for the `server` folder, start the backend:
	npm start
3. Start the Client Server
In a separate terminal for the 'client' folder, start the client:
	npm start
